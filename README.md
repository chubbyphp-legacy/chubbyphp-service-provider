## Description

Pimple service providers.

## Requirements

 * php: ~7.0
 * psr/log: ~1.0

## Suggest

## Installation

Through [Composer](http://getcomposer.org) as [chubbyphp/chubbyphp-service-provider][1].

```sh
composer require chubbyphp/chubbyphp-service-provider "~1.0"
```
